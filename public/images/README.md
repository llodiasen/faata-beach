# Images du projet FAATA Beach

## Image Hero

Pour ajouter l'image de fond de la page d'accueil :

1. Placez votre image dans ce dossier avec le nom : `hero-beach-food.jpg`
2. L'image sera automatiquement utilisée comme fond de la section Hero
3. Format recommandé : JPG ou PNG
4. Résolution recommandée : 1920x1080 ou plus
5. Taille recommandée : < 500KB (optimisée)

## Note

Si l'image n'est pas trouvée, un fond de couleur rouge avec dégradé sera affiché à la place.
